firenodejs™ is a factory robot REST facade that supports one or more of the following actions with individual REST services:

See /camera

firenodejs™ supports up to three cameras with one chosen arbitrarly as the default. Accessing a specific camera by URL (see below) triggers capture of the current image. Video feeds are not supported at this time because current use cases only require still images.
